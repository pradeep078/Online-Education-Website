DSA Array Technical Short Question Answer

Q.1. Which of these best describes an array?

a) A data structure that shows a hierarchical behavior
b) Container of objects of similar types
c) Container of objects of mixed types
d) All of the mentioned

  • B
  • Array contains elements only of the same type.


    • Q.2. How do you initialize an array in C?

    a) int arr[3] = (1,2,3);
    b) int arr(3) = {1,2,3};
    c) int arr[3] = {1,2,3};
    d) int arr(3) = (1,2,3);

  • C
  • This is the syntax to initialize an array in C.


    • Q.3.How do you instantiate an array in Java?

    a) int arr[] = new int(3);
    b) int arr[];
    c) int arr[] = new int[3];
    d) int arr() = new int(3);

  • C
  • Note that option b is declaration whereas option c is to instantiate an array.


    • Q.4. Which of the following is a correct way to declare a multidimensional array in Java?

    a) int[][] arr;
    b) int arr[][];
    c) int []arr[];
    d) All of the mentioned

  • D
  • All the options are syntactically correct.


    • Q.5. What is the output of the following piece of code?

    public class array
    {
      public static void main(String args[])
        {
          int []arr = {1,2,3,4,5};
          System.out.println(arr[2]);
          System.out.println(arr[4]);
        }
    }

    a) 3 and 5
    b) 5 and 3
    c) 2 and 4
    d) 4 and 2

  • A
  • Array indexing starts from 0.


    • Q.6. What is the output of the following piece of code?

    public class array
    {
        public static void main(String args[])
        {
          int []arr = {1,2,3,4,5};
          System.out.println(arr[5]);
        }

    a) 4
    b) 5
    c) ArrayIndexOutOfBoundsException
    d) InavlidInputException

  • C
  • Trying to access an element beyond the limits of an array gives ArrayIndexOutOfBoundsException.


    • Q.7.When does the ArrayIndexOutOfBoundsException occur?

    a) Compile-time
    b) Run-time
    c) Not an error
    d) None of the mentioned

  • B
  • ArrayIndexOutOfBoundsException is a run-time exception and the compilation is error-free..


    • Q.8. Which of the following concepts make extensive use of arrays?

    a) Binary trees
    b) Scheduling of processes
    c) Caching
    d) Spatial locality

  • D
  • Whenever a particular memory location is referred, it is likely that the locations nearby are also referred, arrays are stored as contigous blocks in memory, so if you want to access array elements, spatial locality makes it to access quickly.


    • Q.9.What are the advantages of arrays?

    a) Easier to store elements of same data type
    b) Used to implement other data structures like stack and queue
    c) Convenient way to represent matrices as a 2D array
    d) All of the mentioned

  • D
  • Arrays are simple to implement when it comes to matrices of fixed size and type, or to implement other data structures.


    • Q.10.What are the disadvantages of arrays?

    a) We must know before hand how many elements will be there in the array
    b) There are chances of wastage of memory space if elements inserted in an array are lesser than than the allocated size
    c) Insertion and deletion becomes tedious
    d) All of the mentioned

  • D
  • Arrays are of fixed size, hence during the compile time we should know its size and type, since arrays are stored in contigous locations, insertion and deletion becomes time consuming.


    • Q.11. Assuming int is of 4bytes, what is the size of int arr[15];?

    a) 15
    b) 19
    c) 11
    d) 60

  • D
  • Since there are 15 int elements and each int is of 4bytes, we get 15*4 = 60bytes.
